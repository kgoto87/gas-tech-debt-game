# 技術的負債体感ゲーム

開発プロセスに潜む「技術的負債」を、ゲームを通じて体感するためのWebアプリケーションです。
GAS（Google Apps Script）上で動きます。
GAS での動かし方などは、ここには記載しません。

SocialDog のオフサイトミーティング（[Wantedly の記事はこちら](https://www.wantedly.com/companies/socialdog/post_articles/1057130)）で、Biz を含む全社で技術的負債への理解を深めるアイスブレイクとして使うために作成しました。

## ゲームの概要

チーム対抗で、手話（指文字）を使ってお題（「3文字の動物」など）を当てていくゲームです。
お題をクリアすると文字数分の得点が入りますが、同時に「負債」がランダムに発生します。

負債は **複利のように雪だるま式に増えていく** のがポイントです。

- **継続**：得点を稼ぐ。ただし負債がたまるほどお題が難しくなる（必要な文字数が増える）
- **負債解消**：得点は入らないが、お題をクリアして負債を減らす

得点（デリバリー）と負債解消（リファクタリング）のどちらに投資するか、戦略的なバランスが問われます。

## 技術構成

- Google Apps Script（GAS）の Web アプリとして動作します
- [`Code.gas`](Code.gas)：`doGet()` で [`index.html`](index.html) を配信するエントリーポイント
- [`index.html`](index.html)：ゲーム本体（HTML / CSS / JavaScript）。スコアや負債などの進行状況は `localStorage` にチーム単位で保存されます

## 使い方

1. GAS プロジェクトに `Code.gas` と `index.html`（GASでは `index` という名前のHTMLファイル）を配置する
2. 「ウェブアプリとしてデプロイ」する
3. 発行された URL を開き、チームを選択してゲームを開始する
